# Virkelo Brand Guide

This file defines the canonical brand rules for this repository and its sibling products.

## Brand Architecture

- Umbrella brand: `Virkelo`
- This repository's product name: `Virkelo Field`
- Company behind this repository: `PGP Putki Oy`
- Sister product in the other repository: `Virkelo Assist`
- `Virkelo Assist` modules:
  - `Virkelo Write`
  - `Virkelo Inspect`
  - `Virkelo Lift Advisor`

## What This Repository Is

- This codebase is the field operations product under the Virkelo brand.
- In UI, metadata, copy, and assets, prefer `Virkelo Field`.
- `PGP Putki Oy` is still the operating company/customer context and can appear as the company name.
- Good combined format when needed: `Virkelo Field / PGP Putki Oy`

## Naming Rules

Use these names:

- `Virkelo` for the master brand
- `Virkelo Field` for this app
- `PGP Putki Oy` for company identity

Do not reintroduce these as the main product name:

- `PGP Operointi`
- `PGP Putki Operointi`
- `Putki OS`
- generic `field app` naming when `Virkelo Field` should be shown

If older names exist in technical history, migrations, or comments, do not treat them as the active brand.

## Visual Direction

Primary palette:

- `Virkelo Ink`: `#16324A`
- `Virkelo Blue`: `#2D5C88`
- `Virkelo Mist`: `#EDF4FA`
- `Virkelo Sand`: `#F4EFE6`
- `Virkelo Amber`: `#D38A20`

Design intent:

- industrial, reliable, modern
- premium but practical
- calm blue foundation with warm amber accents
- avoid purple-first branding and generic neon cyber styling

## Logo Assets

Primary mark for this repository:

- `/public/virkelo-mark.svg`

The logo mark is:

- a rounded badge
- a strong `V` monogram
- a flowing amber accent line that suggests movement, direction, and field work

When adding new brand surfaces, prefer the existing SVG instead of drawing a new placeholder mark.

## Copy Guidance

Tone should feel:

- clear
- capable
- grounded
- operational

Prefer wording around:

- field work
- jobs
- reporting
- site photos
- coordination
- command center

Avoid overly playful startup copy when editing operational screens.

## Agent Instructions

When editing this repository:

- preserve `Virkelo Field` as the canonical app name
- preserve the Virkelo palette unless the user asks for a redesign
- use the existing logo asset and brand constants before creating new branding
- keep company references to `PGP Putki Oy` where business context is needed

When editing sibling Virkelo products outside this repo:

- keep `Virkelo` as the umbrella brand
- do not collapse all products back into one unnamed app
